# 题库创建操作说明（简版）

面向运营/非技术同学，快速上手。

## 一、进入后台

1. 启动项目：`npm run dev`
2. 登录管理员账号
3. 访问：`/admin`

## 二、创建题目

1. 进入：`/admin/problems`
2. 填写：
   - 标题、难度（1~10）、标签（逗号分隔）
   - 可见性建议选 `public`
3. 可选填写题面（描述/输入输出/约束/样例）
4. 点击“创建题目”

## 三、给题目添加内容

### 1) 添加新版本

1. 进入题目详情：`/admin/problems/[id]`
2. 填写题面内容 + 时限/内存
3. 点击“创建版本”

### 2) 添加测试点

1. 选择版本
2. 填写输入输出
3. 点击“添加测试点”

### 3) 添加题解

1. 填写题解标题与内容
2. 点击“新增题解”

## 四、导入/导出（批量）

进入：`/admin/import-export`

- JSON 导入：适合完整题库
- CSV 导入：仅基础字段
- ZIP 导入：包含测试点文件
- 导出：可导出 JSON/CSV

## 五、题单（专题）

进入：`/admin/problem-sets`

1. 创建题单（标题/描述）
2. 添加题目（填写题单 ID + 题目 ID + 顺序）

## 六、常见问题

1) 新题目看不到  
- 检查是否登录管理员  
- 可见性是否为 `public`

2) 判题失败  
- 确认已添加测试点  
- 输入输出是否为空
