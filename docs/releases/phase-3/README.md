# Phase 3 README

## 范围

本阶段主要补三类能力：

- 学校 / 机构版 SaaS 打底
- AI 与社区增强
- 平台级学习分析、性能优化与安全加固

## 主要能力

- 租户工作台与开放 API
- AI 推荐、AI 答疑、个性化学习计划
- 社区、小组、动态、积分与奖励
- 平台级学习分析与个性化洞察
- 支付回调、安全校验、限流与查询优化

## 主要前台入口

```text
/tenant
/ai
/discuss
```

## 主要后台入口

```text
/admin/analytics
/admin/organizations
/admin/classes
```

## 核心设计

- 租户隔离以 `Organization` 为逻辑边界，不重做独立多库
- AI 建立在现有学习行为、训练路径、题解与学习报告数据之上
- 社区复用现有 `Post / Comment / Product / Entitlement`
- 平台级分析复用现有提交与进度数据实时聚合
- 安全加固优先修复支付回调、判题回调、内容链接和高频写路径问题

## 本次提交重点

- 机构工作台、开放 API、班级作业与成绩簿
- AI 推荐、AI 辅导、学习计划
- 学习小组、动态分享、积分与兑换
- 平台学习分析、趋势与瓶颈预测
- 支付回调鉴权、判题回调常量时间比较、验证码限流、开放 API 写节流

## 验证重点

- 租户接口的数据隔离
- AI 页面与报告页的联动
- 社区积分发放与兑换一致性
- 平台分析口径与个人学习报告一致
- 支付与判题回调安全校验生效
