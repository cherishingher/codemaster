# 02. 学习端使用说明：题库、提交、学习报告、AI

## 目录

- 题库与题目详情
- 提交与结果查看
- 视频学习
- 图形化练习
- 训练路径
- 学习报告
- AI 推荐、答疑与学习计划

## 题库与题目详情

入口：

```text
/problems
/problems/:id
```

可完成的操作：

- 按关键字、难度、标签浏览题目
- 查看题目描述、输入输出格式、样例、限制条件
- 查看当前题目的题解摘要
- 查看视频解析入口
- 查看相关推荐内容、训练路径与商品导流

题目详情页常见内容：

- 题面
- 提交区
- 题解摘要区
- 完整题解区
- 视频解析区
- 解锁提示区

说明：

- 免费用户默认可看公开内容和高级题解摘要
- 已解锁用户可以查看完整题解和视频解析

## 提交与结果查看

入口：

```text
/submissions
/submissions/:id
```

使用方式：

1. 在题目详情页提交代码
2. 进入提交记录页查看状态
3. 打开提交详情查看编译信息、运行结果、测试点详情

常见结果：

- Accepted
- Wrong Answer
- Time Limit Exceeded
- Runtime Error
- Compile Error

## 视频学习

入口：

```text
/learn
/learn/:slug
```

支持内容：

- 课程列表
- 课时播放
- 视频预览
- 会员内容/已购内容锁态提示

使用方式：

1. 进入视频学习页查看课程
2. 打开课程详情页
3. 对于预览课时可直接观看
4. 对于会员课时或已购课时，按解锁提示操作

## 图形化练习

入口：

```text
/graphical
```

适用场景：

- 面向低门槛可视化练习
- 适合算法启蒙和图形化规则题体验

使用建议：

1. 进入图形化页面选择题目
2. 按页面提示完成拖拽或规则式作答
3. 提交后查看结果并继续迭代

## 训练路径

入口：

```text
/training-paths
/training-paths/:id
/tracks
```

当前支持：

- 入门路径
- 进阶路径
- 高级路径
- 面试准备路径

页面展示内容：

- 路径介绍
- 难度层级
- 章节列表
- 题目状态
- 当前进度
- 最近学习位置
- 锁态与推荐商品

## 学习报告

入口：

```text
/reports/learning
```

基础版内容：

- 最近 7 天活跃天数
- 尝试题数
- 通过题数
- 通过率
- 标签统计
- 当前训练路径进度
- 简单下一步建议

增强版内容：

- 更细的标签表现
- 趋势解读
- 进阶建议
- 需要 VIP 或已解锁权限

数据口径说明：

- 统计窗口默认最近 7 天
- 提交、做题进度、路径推进都基于真实学习数据聚合

## AI 推荐、答疑与学习计划

入口：

```text
/ai
```

可用功能：

- 智能推荐：推荐下一步适合的训练路径、内容或复习方向
- AI 答疑：基于题目、算法、题解上下文进行问答
- 个性化学习规划：按目标生成学习计划

常见使用方式：

1. 先做一批题，积累提交和学习记录
2. 进入学习报告查看薄弱点
3. 打开 AI 页面获取推荐
4. 在题目详情页使用 AI 问答辅助理解
