## 平台适用对象

CodeMaster 适用于以下角色：

- 学生：刷题、学习视频、参加训练营与模拟赛、查看学习报告
- 家长：绑定学生账号，查看增强版学习报告与成长建议
- 教研/运营：维护题解、视频、路径、商品、训练营、模拟赛内容
- 平台管理员：统一管理题库、商品、内容、组织、班级与运营数据
- 学校/机构管理员、教师：在机构工作台中维护班级、成员、作业与统计

## 角色说明

### 游客

- 可以浏览公开页面和公开内容
- 不能提交题目
- 不能查看个人报告、订单、资产等用户数据

### 普通学员

- 可注册并登录
- 可完成基础刷题、查看公开题解摘要、加入社区、查看基础学习报告
- 对会员内容、内容包、训练营、模拟赛等需要单独购买或开通

### VIP 会员

- 可以解锁会员级题解、视频解析、高级训练路径、增强版学习报告
- 到期后权益自动失效，续费后自动顺延

### 已购用户

- 购买内容包、训练营、模拟赛等商品后，可解锁对应资源
- 解锁范围由商品绑定内容决定

### 家长

- 通过家长页绑定学生账号
- 查看学生报告、训练营和模拟赛表现
- 获取复购推荐与成长建议

### 机构管理员/教师

- 进入机构工作台
- 管理组织、班级、学生、教师、题单作业和基础成绩统计

### 平台管理员

- 使用 `/admin` 后台
- 可访问题库、商品、内容、组织、班级、学习分析等模块

## 新用户快速上手

### 第一步：注册或登录

入口：

```text
/register
/login
/forgot-password
```

建议流程：

1. 注册账号
2. 完成登录
3. 进入题库开始做题
4. 查看学习报告与 AI 建议
5. 如需进阶内容，再进入商品中心或会员中心

## 常用页面导航

### 学习主入口

- `/problems`：题库
- `/submissions`：提交记录
- `/learn`：视频学习
- `/training-paths`：训练路径
- `/reports/learning`：学习报告
- `/ai`：AI 辅导

### 付费与成长入口

- `/products`：商品中心
- `/membership`：会员中心
- `/content-packs`：内容包专区
- `/me/orders`：我的订单
- `/me/assets`：我的资产
- `/profile`：个人中心

### 深度训练入口

- `/camps`：训练营
- `/contests`：模拟赛/竞赛
- `/discuss`：社区讨论

### 其他辅助页面

- `/graphical`：图形化练习入口
- `/privacy`：隐私政策
- `/terms`：服务条款
- `/cookies`：Cookie 说明

## 常见使用路径

### 学员路径

1. 注册登录
2. 刷题与提交
3. 查看题解摘要、学习报告
4. 开通会员或购买内容包
5. 进入高级题解、训练路径、训练营、模拟赛

### 家长路径

1. 登录后进入 `/parent`
2. 绑定学生账号
3. 查看成长报告与推荐内容

### 教研/运营路径

1. 登录管理员账号
2. 进入 `/admin/content`
3. 维护题解、视频、路径
4. 发布商品、训练营、模拟赛

### 机构使用路径

1. 平台管理员创建组织
2. 机构管理员进入 `/tenant`
3. 创建班级、导入学生
4. 布置题单并查看统计
