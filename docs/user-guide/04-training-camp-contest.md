# 04. 进阶学习路径、训练营与模拟赛

## 目录

- 进阶学习路径
- 专题训练营
- 模拟赛/竞赛
- 赛后解析与报告
- 推荐使用流程

## 进阶学习路径

入口：

```text
/training-paths
/training-paths/:id
```

平台提供不同难度层次的路径：

- 入门
- 进阶
- 高级
- 面试准备

每条路径包含：

- 适合人群
- 学习目标
- 章节列表
- 章节下题目
- 完成进度
- 最近学习位置

使用建议：

1. 先完成基础刷题
2. 根据当前水平选择路径
3. 每周结合学习报告调整路径推进节奏

## 专题训练营

入口：

```text
/camps
/camps/:id
/camp-class/:id
```

训练营支持能力：

- 训练营列表
- 训练营详情
- 报名入口
- 营期管理
- 名额控制
- 营内班级
- 每日任务
- 学员打卡
- 排行榜
- 结营报告

使用流程：

1. 在训练营列表选择感兴趣的训练营
2. 查看训练营详情：目标、开营时间、适合人群、价格
3. 购买训练营商品完成报名
4. 支付成功后自动入营
5. 进入班级页完成任务、打卡、查看排行和结营报告

## 模拟赛/竞赛

入口：

```text
/contests
/contests/:id
```

当前支持：

- 模拟赛列表
- 比赛详情
- 报名
- 成绩页
- 赛后题解解析
- 赛后报告
- 分组排名报告

说明：

- 本模块是收费模拟赛体系，不是正式 OI/ICPC 赛事系统
- 不包含复杂反作弊、封榜和大规模重判调度

## 赛后解析与报告

赛后内容统一接入权限中心，常见解锁方式：

- 购买模拟赛报名商品后自动获得
- 开通特定会员权益后获得
- 通过活动或赠送获得

赛后可查看内容：

- 赛后题解
- 视频讲评
- 排名表现
- 错题与薄弱点分析

## 推荐使用流程

### 针对进阶用户

1. 选择训练路径
2. 参加专题训练营
3. 参加模拟赛检验阶段成果
4. 查看赛后报告和结营报告
5. 回到学习报告和 AI 页面继续迭代

### 针对冲刺用户

1. 重点参与高级路径和面试准备路径
2. 优先报名模拟赛
3. 使用赛后解析定位问题
4. 配合高级题解与视频解析复盘
