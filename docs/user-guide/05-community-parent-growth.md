# 05. 社区互动、积分奖励与家长端

## 目录

- 学习小组与讨论区
- 动态分享与评论
- 积分与奖励
- 家长绑定与增强报告

## 学习小组与讨论区

入口：

```text
/discuss
/discuss/groups/:id
/discuss/posts/:id
```

当前社区支持：

- 创建学习小组
- 加入学习小组
- 在小组内发帖讨论
- 发布学习动态、成就和问题
- 评论互动

适合讨论的话题：

- 课程内容
- 题目思路
- 学习技巧
- 模拟赛复盘
- 训练营打卡分享

## 动态分享与评论

社区动态可用于：

- 分享最近完成的学习路径
- 分享拿到的 AC、排名、训练营阶段成果
- 提出学习问题
- 邀请其他同学讨论某个题目或算法主题

建议发布内容：

- 近期进展
- 遇到的难点
- 想请教的问题
- 对某条题解或视频的心得

## 积分与奖励

社区与学习行为会产生积分，当前常见积分来源包括：

- 创建学习小组
- 加入学习小组
- 发布动态/帖子
- 评论互动
- 首次通过一道题

积分用途：

- 兑换商品
- 兑换课程或内容包
- 作为社区活跃度激励

查看入口：

- 社区首页会展示积分余额、积分明细和最近兑换

说明：

- 可兑换商品通常由后台配置积分价格
- 商品兑换后仍然通过现有商品/权益系统完成解锁

## 家长绑定与增强报告

入口：

```text
/parent
```

家长端支持：

- 绑定学生账号
- 切换查看不同学生
- 查看增强版学习报告
- 查看训练营、模拟赛和整体成长表现
- 查看推荐商品和下一步建议

轻量绑定流程：

1. 家长登录
2. 在家长页输入学生邮箱或手机号
3. 建立绑定关系
4. 在家长页切换学生查看报告

注意事项：

- 家长只能看到已绑定学生的数据
- 家长报告是基于学生真实学习数据聚合得出
