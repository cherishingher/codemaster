# 03. 商品、订单、会员与内容解锁

## 目录

- 商品中心
- 商品详情与下单
- 支付与订单
- 我的资产
- 会员中心
- 内容包与统一解锁规则

## 商品中心

入口：

```text
/products
/products/:id
```

支持商品类型：

- 会员商品
- 训练路径商品
- 内容包商品
- 训练营商品
- 模拟赛报名商品

列表页展示信息：

- 商品名称
- 标签
- 封面
- 价格与原价
- 权益说明
- 已购/未购状态

## 商品详情与下单

商品详情页会展示：

- 商品简介
- 详细描述
- SKU 选择
- 价格、原价
- 权益列表
- 包含内容
- 当前用户是否已解锁

典型流程：

1. 进入商品详情页
2. 选择 SKU
3. 点击去结算
4. 在结算页确认订单
5. 进入支付流程

## 支付与订单

入口：

```text
/checkout
/payments/:paymentNo
/payments/success
/payments/failed
/me/orders
/orders/:id
```

当前能力：

- 创建订单
- 创建支付单
- mock 支付
- 支付回调
- 我的订单列表
- 订单详情

订单常见状态：

- `CREATED`
- `PENDING`
- `PAID`
- `COMPLETED`
- `CLOSED`
- `REFUNDING`
- `REFUNDED`

说明：

- 支付回调支持幂等处理
- 支付成功后会自动发放相应权益
- 用户刷新页面后，订单状态和解锁状态会保持正确

## 我的资产

入口：

```text
/me/assets
```

用于查看：

- 已购买商品
- 当前生效权益
- 到期时间
- 资产汇总

适合用于确认：

- 某个内容包是否已解锁
- 训练营/模拟赛报名是否生效
- 会员是否处于有效期内

## 会员中心

入口：

```text
/membership
/membership/benefits
```

当前会员策略：

- 单层会员：VIP
- 常见 SKU：月卡、季卡、年卡

会员中心可查看：

- 当前状态
- 到期时间
- 剩余时长
- 当前已生效权益
- 续费入口

续费规则：

- 当前仍有效：从当前到期时间继续顺延
- 已过期：从本次购买时间重新开始

## 内容包与统一解锁规则

入口：

```text
/content-packs
```

统一权限来源：

- FREE：免费
- MEMBERSHIP：会员解锁
- PURCHASE：购买解锁
- GIFT：赠送解锁
- ACTIVITY：活动解锁

统一判断原则：

- 后端负责权限校验
- 未解锁时只返回摘要和锁态信息
- 已解锁时返回完整内容
- 前端只做展示，不提前拿完整内容

常见锁态提示：

- 需要 VIP
- 需要购买
- VIP 或购买均可
- 先登录后查看
