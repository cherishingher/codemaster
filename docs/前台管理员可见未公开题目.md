# 前台管理员可见未公开题目（功能提示）

## 功能目的
- 让管理员在**前台**直接查看尚未公开的题目，便于自测与审核。
- 普通用户仍只看到公开题目，避免误曝光。

## 可见规则
- **未登录 / 普通用户**：仅返回 `visibility = "public"` 的题目。
- **管理员（roles 含 admin）**：可返回所有题目（包含未公开）。
- 未公开题目对普通用户访问会返回 404。

## 影响范围
- 接口：
  - `GET /api/problems`：管理员可看到全部题目。
  - `GET /api/problems/[id]`：管理员可查看未公开题目详情。
  - 两个接口都会返回 `visibility` 字段。
- 前端：
  - `/problems` 列表页对未公开题目显示「未公开」标识。
  - `/problems/[id]` 详情页对未公开题目显示「未公开」标识。

## 使用流程（管理员）
1. 登录管理员账号（`roles` 包含 `admin`）。
2. 打开 `/problems` 列表，可看到未公开题目并带「未公开」标识。
3. 点击题目进入详情页进行自测或检查内容。
4. 确认无误后，在管理端把题目设为公开。

## 常见问题
- **列表为空**：可能未登录管理员，或系统中没有公开题目。
- **普通用户能否访问未公开题目**：不能，接口返回 404。
- **要验证普通用户视角**：使用无痕窗口或退出登录。
